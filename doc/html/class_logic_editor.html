◆ addInv()

void LogicEditor::addInv ( )

Добавляет или удаляет инверсию для выделенных ячеек в таблице формул. Если таблица формул не инициализирована, функция ничего не делает.

Функция проверяет количество строк и колонок в таблице формул. Если количество строк меньше 2 или количество колонок равно 0, функция завершает свою работу. Затем происходит итерация по ячейкам таблицы формул. Если ячейка существует и выделена, проверяется текущий цвет фона. Если цвет черный, это означает, что в ячейке уже есть инверсия, и она удаляется, устанавливая белый цвет фона. Если цвет не черный, то инверсия отсутствует, и она добавляется, устанавливая цвет фона ячейки в черный. По окончании процесса изменения цветов в ячейках, функция отправляет сигнал onStatusChanged() для указания, что произошли изменения.

◆ clearTable()

void LogicEditor::clearTable ( )

Очищает таблицу редактора логики. Если указатель formulaTableWidget равен nullptr, ничего не делает. Иначе производит следующие действия: Получает количество строк и столбцов таблицы. Перебирает все ячейки таблицы и устанавливает пустую строку в каждую ячейку. Освобождает память, выделенную для каждой ячейки таблицы. Устанавливает количество строк и столбцов таблицы в 0. Форматирует таблицу заново. Отправляет сигнал о изменении статуса.

◆ deleteLastSymvol()

void LogicEditor::deleteLastSymvol ( )

Удаляет последний символ в таблице редактора логики. Если указатель formulaTableWidget равен nullptr, ничего не делает. Иначе производит следующие действия: Получает количество строк и столбцов таблицы. Если количество столбцов равно 0, ничего не делает. Если режим редактирования выбранных включен:

  • Удаляет выделенные столбцы из таблицы. В противном случае:
  • Устанавливает пустую строку в последнюю ячейку каждой строки.
  • Освобождает память, выделенную для последней ячейки каждой строки.
  • Уменьшает количество столбцов таблицы на 1. Форматирует таблицу заново. Отправляет сигнал о изменении статуса.

◆ formatTable()

void LogicEditor::formatTable ( )

Форматирует таблицу в редакторе логики.

Если таблица не существует или не содержит ни одной строки, то создает 5 строк. Если таблица содержит строки, но не содержит столбцов, то возвращает управление. Для каждой ячейки таблицы:

  • Если ячейка пустая, выделяет память, устанавливает белый фон и шрифт Segoe UI размером 13.
  • Если ячейка не пустая, удаляет текст ячейки. Устанавливает ширину столбцов и высоту строк.
